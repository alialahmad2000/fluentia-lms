## PART 2: UNIVERSAL UNIT TEMPLATE

Every Reading Explorer unit follows this **identical structure** across all 6 levels:

### Unit Structure (13-14 pages per unit)

| Section | Page(s) | Description |
|---------|---------|-------------|
| **Unit Opener** | 1 page | Full-page photo with theme title, caption, WARM UP box (2 discussion questions with a partner) |
| **1A: Before You Read** | 1 page | Exercise A (labeled skill: DEFINITIONS/PREVIEWING/DISCUSSION) + Exercise B (labeled skill: SKIMMING/SCANNING) + large photo with caption containing bold target words |
| **1A: Reading Passage** | 1-3 pages | Title in large display font, paragraphs labeled A-E (Foundation) to A-L (Level 5), bold/colored vocabulary words, footnotes, photos/illustrations |
| **1A: Infographic/Visual** | 1 page | Data visualization, map, diagram, photo essay, or chart related to passage topic |
| **1A: Reading Comprehension** | 1 page | Section A: 5 MCQs (labeled by type: GIST, DETAIL, INFERENCE, VOCABULARY, REFERENCE, PURPOSE, PARAPHRASE, SEQUENCE, MAIN IDEA, SCANNING) + Section B: summary/completion/matching activity |
| **1A: Reading Skill** | 1 page | Skill explanation box + Exercise A (labeled skill) + Exercise B (labeled skill) + CRITICAL THINKING box (partner discussion) |
| **1A: Vocabulary Practice** | 1 page | Exercise A (DEFINITIONS/COMPLETION—mini reading passage) + Exercise B (DEFINITIONS/COMPLETION) + Exercise C (WORD FORMS/WORD USAGE/COLLOCATIONS) |
| **1B: Before You Read** | 1 page | Same structure as 1A but different skill labels (PREVIEWING/SCANNING/SKIMMING/DISCUSSION) |
| **1B: Reading Passage** | 1-3 pages | Same structure, sometimes slightly different length |
| **1B: Reading Comprehension** | 1 page | Same structure as 1A comprehension |
| **1B: Reading Skill** | 1 page | Same structure + CRITICAL THINKING at bottom |
| **1B: Vocabulary Practice** | 1 page | Same 3-exercise structure |
| **Video Section** | 2 pages | BEFORE YOU WATCH (Exercise A: vocabulary preview + Exercise B: discussion/prediction) → WHILE YOU WATCH (GIST MCQ + DETAILS completion) → CRITICAL THINKING + VOCABULARY REVIEW (checklist of unit words) |

**Total per unit:** 13-14 pages (Foundation) to 17-18 pages (Level 5, due to longer passages)

### 6.3 Unit Structure Template

**Left Page (Explanation):**
- Unit number and title in header (e.g., "Unit 1: Present continuous (I am doing)")

- Rule box with grammar pattern (colored/highlighted)
- Conjugation tables where relevant
- Multiple example sentences (typically 6-12)
- Comparison boxes (e.g., "I am doing" vs "I do")
- Common mistakes or notes in separate boxes
- Cross-references to related units
- English-only explanations (no L1 translation)

**Right Page (Exercises):**
- Typically 4 exercises per unit
- Exercise types observed:
  - Fill-in-the-blank (most common)
  - Choose the correct form
  - Rewrite sentences
  - Complete using pictures/illustrations
  - Match sentence halves
  - Error correction (find and fix mistakes)
  - Write sentences from prompts
  - Put verbs in correct form
- Items per exercise: typically 6-10
- Progressive difficulty within the page (guided → semi-free)
- Total items per unit: approximately 25-35

### 6.5 Grammar Progression Map

The grammar builds from **concrete → abstract** and **simple → complex**:

1. **Foundation Layer** (Units 1-18): All tenses (present, past, present perfect, past perfect) — the backbone of English grammar
2. **Future Layer** (Units 19-25): Future forms and time clauses
3. **Modal Layer** (Units 26-37): Ability, obligation, permission, probability — adding nuance
4. **Conditional Layer** (Units 38-41): Hypothetical thinking
5. **Voice Layer** (Units 42-46): Passive — changing perspective
6. **Reporting Layer** (Units 47-48): Reported speech
7. **Question Layer** (Units 49-52): Question formation and tags
8. **Verb Pattern Layer** (Units 53-68): -ing/to patterns — the most complex verb area
9. **Noun System** (Units 69-81): Articles, countability, noun phrases
10. **Pronoun/Determiner Layer** (Units 82-91): Reference and quantity
11. **Clause Layer** (Units 92-97): Relative clauses — complex sentences
12. **Modifier Layer** (Units 98-111): Adjectives, adverbs, comparison
13. **Connector Layer** (Units 112-120): Conjunctions, linking ideas
14. **Preposition Layer** (Units 121-136): Position, time, verb/adj + preposition
15. **Phrasal Verb Layer** (Units 137-145): The most idiomatic area

## PART 7: PEDAGOGICAL PATTERNS

### 7.1 Lesson Flow Model

Each RE unit follows a consistent **Activate → Input → Process → Practice → Produce** cycle, repeated twice (for Reading A and B), then capped with a Video section:

```
UNIT OPENER (Activate)
  └── Full-page photo + Warm Up discussion → activates schema, builds interest

READING A CYCLE:
  ├── BEFORE YOU READ (Activate + Pre-teach)
  │   ├── Exercise A: Pre-teach key vocabulary through definitions/context
  │   └── Exercise B: Set purpose for reading (skim/scan/predict)
  │
  ├── READING PASSAGE (Input)
  │   └── Authentic-style text with embedded vocabulary
  │
  ├── INFOGRAPHIC (Input + Visual Literacy)
  │   └── Data/visual connected to passage topic
  │
  ├── READING COMPREHENSION (Process)
  │   ├── Section A: MCQ — test understanding
  │   └── Section B: Summary/completion — synthesize
  │
  ├── READING SKILL (Process + Explicit Instruction)
  │   ├── Explanation box — teach the strategy
  │   ├── Exercise A — guided practice of strategy
  │   ├── Exercise B — semi-guided practice
  │   └── CRITICAL THINKING — apply/evaluate/create
  │
  └── VOCABULARY PRACTICE (Practice)
      ├── Exercise A: Definitions in new context
      ├── Exercise B: Sentence-level practice
      └── Exercise C: Word forms/collocations/usage

READING B CYCLE: [same structure repeated]

VIDEO (Produce + Extend)
  ├── BEFORE YOU WATCH: Preview vocabulary + predict
  ├── WHILE YOU WATCH: Comprehension check
  ├── CRITICAL THINKING: Discussion/reflection
  └── VOCABULARY REVIEW: Self-assessment checklist
```

### 7.2 Vocabulary Encounter Pattern

Each target word gets **minimum 4-5 encounters** per unit:

| Encounter | Stage | Type | Activity |
|-----------|-------|------|----------|
| 1 | Before You Read | Receptive | See word in bold in photo caption or exercise |
| 2 | Reading Passage | Receptive | See word in bold within authentic context |
| 3 | Reading Comprehension | Receptive/Productive | Word may appear in MCQ questions/answers |
| 4 | Vocabulary Practice A | Productive | Use word in new context (mini passage) |
| 5 | Vocabulary Practice B/C | Productive | Practice word forms, collocations, usage |
| 6 | Vocabulary Review (Video) | Self-assessment | Check if word is known; review if not |

**Recycling pattern:** Words are NOT explicitly recycled across units within a single book. Each unit has its own word set. However, **word families and collocations** build on each other (e.g., suffixes taught in U3 are used in U6's vocabulary).

### 7.3 Scaffolding Analysis

**Within each section (micro-scaffolding):**
- Before You Read: **Guided** (definitions given) → **Semi-guided** (skim with question)
- Comprehension: **Controlled** (MCQ—recognition) → **Semi-controlled** (summary completion—production)
- Reading Skill: **Explicit instruction** (explanation box) → **Guided practice** (Exercise A) → **Independent practice** (Exercise B) → **Transfer** (Critical Thinking)
- Vocabulary: **Recognition** (Exercise A—definitions) → **Production** (Exercise B—sentences) → **Extension** (Exercise C—word forms/collocations)

**Within each unit (macro-scaffolding):**
- Reading A is always slightly easier than Reading B (shorter, simpler vocabulary)
- Reading Skill A teaches a strategy; Reading Skill B may extend or teach a complementary strategy
- Critical Thinking at the end of each half-unit is always the most open-ended, least scaffolded activity

**Across levels (cross-level scaffolding):**
- Same unit structure maintained → students know what to expect
- Passage length increases gradually (~250 → ~1200 words)
- MCQ choices increase (3 → 4) to increase difficulty
- Question types add new categories at higher levels
- Vocabulary exercises add morphological complexity (word roots, affixes)
- Critical Thinking moves from concrete (apply, categorize) to abstract (evaluate, synthesize)
