## A.6 — Unit Content Page Integration

**Unit content page file:** `src/pages/student/curriculum/UnitContent.jsx`

**Tab switching mechanism:** URL-based `?activity=grammar` param. The page uses `const activeActivity = searchParams.get('activity') || null`. Tabs are lazy-loaded via `React.lazy()`:
```javascript
const GrammarTab = React.lazy(() => import('./tabs/GrammarTab'))
```

**Lazy-mount-once pattern:** Tabs stay mounted after first open via `visitedTabs` Set (line 123):
```javascript
useEffect(() => {
  if (activeActivity) {
    setVisitedTabs(prev => { /* add to set */ })
  }
}, [activeActivity])
```

Tabs render with: `{(activeActivity === 'grammar' || visitedTabs.has('grammar')) && <GrammarTab unitId={unit?.id} />}`

**Hooks concern for injection:** `GrammarTopic` (the component receiving the injection) has ALL hooks at the top — no violation risk. The `DialectExplanationCard` must be placed inside `GrammarTopic`'s return, not at the `GrammarTab` level.
